# Matrix
A lightweight C++ template library for matrix operations with STL-compatible iterators.

# Матричная библиотека на C++

## О проекте

В качестве пет-проекта был реализован RAII-класс для работы с матрицами на языке C++.

Проект создавался с целью:
- Изучить управление памятью в C++
- Освоить идиому RAII
- Реализовать правило пяти
- Сделать шаблонный класс доступным для алгоритмов STl
- Применить исключения для обработки ошибок
- Реализовать прокси-класс

## Что такое RAII

RAII (Resource Acquisition Is Initialization) — это фундаментальная идиома C++, которая означает, что управление ресурсами привязано к времени жизни объекта.

В переводе на русский: **Захват ресурса есть инициализация**.

Простыми словами: ресурс (в данном случае память для хранения элементов матрицы) захватывается в конструкторе и освобождается в деструкторе. Пользователю не нужно вручную вызывать `delete[]` — деструктор сработает автоматически, когда объект выйдет из области видимости.


Это гарантирует отсутствие утечек памяти и безопасность даже при возникновении исключений.

## Прокси-класс

Помимо основного RAII-класса, в проекте реализован **прокси-класс ProxyRow**. Это вспомогательный внутренний класс, который используется для реализации оператора `[][]`. Он хранит указатель на строку матрицы и позволяет обращаться к отдельным элементам, как если бы матрица была двумерным массивом. При этом прокси-класс также проверяет выход за границы столбцов и выбрасывает исключение `std::out_of_range`, что делает доступ к элементам безопасным.

```cpp
// Пример работы прокси-класса
Matrix<int> m(2, 3);
m[0][1] = 42;  // Прокси-класс позволяет использовать двойной оператор []
```

## Итераторы

Для интеграции с алгоритмами стандартной библиотеки в классе Matrix реализованы **forward-итераторы**. Они основаны на категории `std::forward_iterator_tag` и поддерживают однонаправленный обход элементов матрицы.

Особенностью реализации является **наследование итераторов**: неконстантный итератор `iterator` наследуется от константного `const_iterator`. Это позволяет:
- Избежать дублирования кода
- Обеспечить автоматическое преобразование `iterator` в `const_iterator` там, где это необходимо
- Поддерживать константную корректность

Реализованы как константная, так и неконстантная версии итераторов, что обеспечивает безопасный доступ к элементам матрицы как для чтения, так и для записи.

Итераторы позволяют:
- Использовать матрицу в range-based for
- Применять алгоритмы STL (`std::find`, `std::transform`, `std::accumulate` и другие)
- Передавать итераторы в стандартные контейнеры и функции

```cpp
// Пример использования итераторов
Matrix<int> m(2, 3, {1, 2, 3, 4, 5, 6});

// Range-based for
for (int& x : m) {
    x *= 2;
}

// STL алгоритмы
int sum = std::accumulate(m.begin(), m.end(), 0);
auto it = std::find(m.begin(), m.end(), 10);
```

## Ключевые особенности

### Управление памятью

Библиотека полностью реализует **правило пяти**:
- Конструктор копирования — создаёт глубокую копию матрицы
- Оператор копирующего присваивания
- Конструктор перемещения — эффективно передаёт владение данными
- Оператор перемещающего присваивания
- Деструктор — автоматически освобождает память

Это гарантирует корректное управление динамической памятью и предотвращает утечки.

### Безопасность и контроль ошибок

- Выход за границы матрицы контролируется исключениями
- Конструктор от потока ввода выбрасывает исключение при ошибке чтения
- Операции определителя и следа проверяют, что матрица квадратная
- Тип данных ограничен арифметическими типами

## Конструкторы

Матрицу можно создать несколькими способами:

**1. Заполнить все элементы значением:**
```cpp
Matrix<int> m1(2, 3, 5);
```

**2. Из списка инициализации:**
```cpp
Matrix<int> m2(2, 3, {1, 2, 3, 4, 5, 6});
```

**3. Из диапазона итераторов:**
```cpp
std::vector<int> data = {1, 2, 3, 4, 5, 6};
Matrix<int> m3(2, 3, data.begin(), data.end());
```

**4. Из потока ввода:**
```cpp
std::istringstream iss("1 2 3 4 5 6");
Matrix<int> m4(iss, 2, 3);
```

**5. Единичная матрица:**
```cpp
Matrix<int> I1 = Matrix<int>::eye(3);      // 3×3
Matrix<int> I2 = Matrix<int>::eye(2, 3);   // 2×3
```

## Пример использования

```cpp
#include <iostream>
#include "matrix.hpp"

int main() {
    // Создание матриц
    Matrix<int> A(2, 3, {1, 2, 3, 4, 5, 6});
    Matrix<int> B(3, 2, {7, 8, 9, 10, 11, 12});
    
    // Умножение матриц
    Matrix<int> C = A.multiply(B);
    C.dump(std::cout);
    
    // Транспонирование
    C.transpose();
    C.dump(std::cout);
    
    // Range-based for
    for (int& x : C) {
        x *= 2;
    }
    
    // STL алгоритмы
    int sum = std::accumulate(C.begin(), C.end(), 0);
    auto it = std::find(C.begin(), C.end(), 10);
    
    // Определитель
    Matrix<int> D(3, 3, {1, 2, 3, 4, 5, 6, 7, 8, 10});
    int det = D.determinant();
    
    std::cout << "Определитель: " << det << "\n";
    
    return 0;
}
```

- Стандартная библиотека C++ (STL)
